## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies:
```
npm install
```

- Run the development web server:
```
npm start
```

- Run TinyApp on your browser:
```
http://localhost:8080/
```
